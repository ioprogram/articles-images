# Assets per la Guida Java SE 25

Repository pubblico contenente immagini, diagrammi e risorse visive per la serie di articoli su Java SE 25.

## 📁 Struttura

Ogni articolo ha la propria cartella contenente tutte le risorse visive associate:

```
articles-images/
├── articolo-01/
│   ├── lexical_translation.png
│   └── token_classification.png
├── articolo-02/
│   └── ...
└── ...
```

## 🔗 Utilizzo

Le immagini sono accessibili pubblicamente tramite GitHub raw URL:

```
https://raw.githubusercontent.com/ioprogram/articles-images/main/articolo-XX/nome-immagine.png
```

## 📝 Note

Questo repository è complementare al repository privato [jvm-articles](https://github.com/ioprogram/jvm-articles) che contiene gli articoli completi.
